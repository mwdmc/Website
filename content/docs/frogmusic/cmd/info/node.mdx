## 指令簡介
<Info>
查看音樂節點資訊，並且節點的狀態。
</Info>

## 指令需求
| 需要 DJ |	需要在頻道 | 需要正在播放 |
| :-----: | :-------: | :---------: |
|   ❎   |    ❎     |     ❎      |

## 使用方法
打出 `/node` 即可顯示節點資訊及狀態。

## 顯示的東西
- 運行狀態
- 節點名稱
- 節點播放器使用量
- 節點在線時間
- 節點核心以及記憶體使用量
- 系統及音樂負載資訊
