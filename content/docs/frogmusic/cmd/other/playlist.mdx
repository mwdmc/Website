## 指令簡介
<Info>
個人專屬播放清單
</Info>

## 指令需求
| 需要 DJ |	需要在頻道 | 需要正在播放 |
| :-----: | :-------: | :---------: |
|   ❎   |    ❎     |     ❎      |

## 使用方法
### 新增音樂至個人化播放清單
- /playlist add `播放清單名稱` `要新增的歌曲名稱`
### 重新命名個人化播放清單
- /playlist rename `播放清單名稱` `新的播放清單名稱`
### 從個人化播放清單中移除音樂
- /playlist remove `播放清單名稱` `歌曲編號`
- 可以透過 `/playlist info` 取得歌曲編號
### 顯示個人化播放清單資訊
- /playlist info `播放清單名稱`
### 建立個人化播放清單
- /playlist create `要建立的播放清單名稱`
### 刪除個人化播放清單
- /playlist delete `要刪除的播放清單名稱`
### 列出個人化播放清單
- /playlist list
### 播放個人化播放清單
- playlist play `播放清單名稱`

## 顯示的東西
### /playlist add
- 請提供播放清單名稱
- 找不到播放清單
- 請提供歌曲
- 搜尋歌曲時發生了錯誤，請稍後再試
- 歌曲已經在播清單中
- 已將歌曲加入播放清單
### /playlist rename
- 請提供播放清單名稱
- 請提供新的播放清單名稱
- 播放清單名稱只能有 50 個字元
- 找不到播放清單
- 此播放清單名稱已經存在
- 已重新命名播放清單
### /playlist remove
- 請提供播放清單名稱
- 請提供歌曲編號
- 找不到播放清單
- 在此播放清單中找不到這個歌曲編號
- 已從此播放清單移除歌曲
### /playlist info
- 找不到播放清單
- 播放清單中沒有歌曲
### /playlist create
- 播放清單已經存在
- 您已達到最大播放清單數量
- 已成功創建播放清單
### /playlist delete
- 找不到播放清單
- 已刪除播放清單
### /playlist list
- 您沒有任何播放清單
### /playlist play
- 已將此播放清單加入待播清單
- 找不到播放清單
- 這個播放清單是空的
- 必須在語音頻道中才能使用此指令
